# Memory Enhancement Plugin (Tables)

This plugin enhances the memory capabilities of SillyTavern by using structured tables to organize and store character information, events, and other important data.

## Features
-   **Structured Memory**: Automatically organizes chat information into tables (e.g., Character Features, Important Events, Inventory).
-   **Auto-Update**: Updates tables based on new chat messages.
-   **Customizable**: Modify table structures and rules.
-   **Visualization**: View memory tables in an easy-to-read format.

## Installation
1.  Navigate to the `extensions` directory in your SillyTavern installation.
2.  Clone this repository: `git clone https://github.com/chaaruze/st-memory-enhancement`
3.  Restart SillyTavern.

## Usage
-   The plugin adds an "Event table" icon to the top bar.
-   Click the icon to view and manage tables.
-   Settings can be accessed via the extension drawer (wrench icon).
